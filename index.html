    Capitulo 01

    Bem vindo
    Esta atividade foi desenvolvida com o objetivo de exercitar os fundamentos essenciais da linguagem HTML5, incluindo a criação e estruturação básica de documentos web. Durante a prática, foram utilizados elementos semânticos fundamentais como: <html>, <head>, <body> para a construção da estrutura básica de uma página HTML; <title> para definição do título da aba no navegador; <h1> a <h6> para hierarquia de títulos e organização de conteúdo; <p> para parágrafos de texto; <a> para criação de hiperlinks; <img> para inserção de imagens; <ul>, <ol>, e <li> para listas' ordenadas e não ordenadas.
    Tabelas
    Esta atividade teve como foco o aprofundamento nos elementos de marcação HTML voltados para a apresentação tabular de dados e a manipulação de imagens dentro de páginas web. Foram utilizados os seguintes elementos semânticos: <table> para declarar a estrutura da tabela; <thead>, <tbody> e <tfoot> para organizar logicamente o conteúdo da tabela (cabeçalho, corpo e rodapé); <tr> para definir linhas; <th> para células de cabeçalho com significado semântico e visual diferenciado (geralmente negrito e centralizado); <td> para células de dados regulares. A atividade incluiu também a aplicação de atributos como: colspan e rowspan para mesclar células horizontal ou verticalmente;
    Listas
    Esta atividade foi desenvolvida com o objetivo de praticar o uso da estrutura semântica de listas de definições (<dl>, <dt>, <dd>) em conjunto com a criação de links internos na mesma página utilizando âncoras nomeadas (id e <a href="#...">). Lista de Definições A estrutura foi composta por: <dl> para encapsular a lista como um todo; <dt> para representar os termos a serem explicados; <dd> para fornecer a descrição ou definição associada a cada termo. Cada <dt> representava um conceito, enquanto as respectivas <dd> ofereciam explicações detalhadas e exemplos práticos. Links Internos (âncoras) Para melhorar a navegação e a organização da informação na página: Foram adicionados links no topo da página que direcionam o usuário diretamente para cada definição específica; Cada <dt> recebeu um atributo id único, e os links foram criados usando a tag <a href="#id">
    Formularios
    Esta atividade foi desenvolvida com o objetivo de praticar a construção de formulários utilizando os elementos semânticos e atributos próprios do HTML5, voltados para a coleta de dados de forma estruturada e acessível. Elementos do Formulário A estrutura foi composta por: <form> para encapsular o conjunto de campos e definir o destino dos dados (via atributo action); <input> para entrada de dados, incluindo os tipos text, email, number, password, radio, checkbox e submit; <label> para associar descrições textuais aos campos, melhorando a acessibilidade; <textarea> para inserção de textos longos; <select> e <option> para criação de listas suspensas de opções. Também foram aplicados atributos HTML5 como: required para campos obrigatórios; placeholder para dicas dentro dos campos; min, max e maxlength para controle de validação de dados; name e id para identificação dos campos.

    Capitulo 2

    Primeiros passos em CSS
    Esta atividade teve como objetivo apresentar os fundamentos da linguagem CSS (Cascading Style Sheets), utilizada para aplicar estilos visuais a documentos HTML, promovendo a separação entre estrutura (HTML) e apresentação (CSS).
    Criação de bordas
    Esta atividade teve como foco a aplicação prática de estilos visuais com CSS por meio da definição de bordas e cores em elementos HTML. O objetivo foi explorar as principais propriedades responsáveis pela aparência visual de blocos de conteúdo, melhorando a organização e a estética da interface.
    Links estilizados
    A atividade focou na personalização de links com CSS, alterando aparência padrão dos elementos <a>. Foram aplicadas propriedades como color, text-decoration, hover, visited e active para controlar estilo e interatividade.
    Listas estilizadas
    Esta atividade teve como foco personalizar listas HTML (<ul>, <ol>, <li>) com CSS. Foram utilizados estilos para remover marcadores, alterar espaçamentos, cores e aplicar ícones personalizados.
    Formularios estilizados
    A atividade abordou a personalização visual de formulários HTML usando CSS. Foram aplicados estilos aos campos <input>, <textarea> e <button>, com foco em cores, bordas, espaçamento e responsividade.
    Tabelas estilizadas
    A atividade consistiu em aplicar estilos visuais em tabelas HTML utilizando CSS. Foram ajustadas propriedades como bordas, espaçamento, cores de fundo e alinhamento de texto para melhorar a legibilidade.
    Float
    A atividade teve como foco o uso da propriedade float do CSS para posicionar elementos lado a lado. Foram testados valores como left, right.

    Capitulo 3

    Portal da Floresta
    A atividade explorou o uso das principais tags semânticas do HTML5 para estruturar páginas web de forma organizada e acessível. Foram utilizadas header, nav, main, aside e footer para representar seções distintas do conteúdo.
